datetime-local : 사용자가 있는 지역을 기준으로 날짜와 시간

date : 사용자가 있는 지역을 기준으로 날짜(연, 월, 일)

month : 사용자 지역을 기준으로 날짜(연, 월)

week : 사용자 지역을 기준으로 날짜(연, 주)

time : 사용자가 있는 지역을 기준으로 시간(시, 분, 초, 분할초)

button : 버튼 삽입
