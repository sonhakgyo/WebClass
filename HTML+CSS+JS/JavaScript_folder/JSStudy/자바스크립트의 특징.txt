1장. 자바스크립트의 특징
1. 자바스크립트의 특징
- 자바스크립트는 네스케이프 회사에서 개발한 HMTL 문서 내에 삽입되어 동적인 웹페이즈를 구현할 수 있는 스크립트이다.
- 독립된 응용프로그램의 형태를 지니지 못하는 스크립트 언어다(완벽한 문법구조가 아니다).
- 완벽한 문법적인 구조를 가지지 못하기 때문에 스크립트 언어라고 한다.
- 객체를 이용하여 프로그래밍하는 객체기반 언어이다. → 객체지향언어
- 브라우저에 대한 우수한 호환성으로 주로 Client-side(익스플로러)에서 프로그래밍하는 스크립트로 사용된다. → 서버 프로그래밍에서 더 공부할 수 있다.
- HTML 안에 삽입될 때는 <script>내용</script> 안에 정의한다.

2. 자바스크립트를 정의하는 방법
- <script>내용</script> 안에 정의한다.
- <script>내용</script>의 위치는 '<head>, </head>' 태그 또는 
'<body>, </body>' 태그 안에 모두 정의할 수 있다. 
- 그러나, 번역기에 의해서 먼저 번역되길 권장하기 때문에 보통 <head>, </head> 태그 안에 정의하길 권장한다.

4. 자바스크립트의 자료형
- 자바스크립트의 자료형은 varient(뜻:다양한) 타입이라고 할 수 있다.
- 세부적으로는 char, int, double, float, int, long, short, void 등의 타입이 있다.
- 자료형을 엄격히 따지지 않는 스크립트 언어이므로 a=10과 같이 처음부터 치환문을 이용해도 된다.

5. 변수 만드는 규칙 
- 사용 가능한 문자는 영문자, 숫자, _(underbar, underscore)이다.
- 숫자로 시작할 수 없다.
- _(underbar, underscore)를 제외한 !, #, ?, $, @ 등과 같은 다른 특수문자는 올 수 없다.
- 공백은 허용되지 않는다.
- 자바스크립트 문법 안에서 규정한 예약어(키워드)는 변수로 사용할 수 없다.
    <변수 작성의 예>
        01. Abc (O)
        02. 123abc (X)  ← 숫자로 시작할 수 없다.
        03. Total# (X)  ← 특수문자 #은 들어갈 수 없다.
        04. Two of (X)  ← 공백(띄어쓰기)은 들어갈 수 없다.
        05. int (X)     ← int는 예약어(키워드)다.
        06. Not! (X)    ← 특수문자 !는 들어갈 수 없다.
        07. _tot (O)     


2장. 자바스크립트 연산자(산술, 대입, 증감, 비교연산자)
i. 산술연산자 : +, -, *, /, %(나머지 구하는 연산자)

iii. 증감연산자 : 전위 연산자 - 피연산자의 앞쪽에 위치한다.
               후위 연산자 - 피연산자의 뒤쪽에 위치한다.
4. 비교연산자 : ==(같다), !=(같지 않다)
