예외처리(Exception)
    에러와 예외는 다른 말이다. 둘의 구분선은 애매하다.
        에러 : 심각한 오류
        예외 : 덜 심각한 오류



예외처리 방법 
    (1) 방법.
    try {
            오류가 발생할 수 있는 문장
    } catch (예외이름 객체명) {          
            오류 발생 시 실행할 문장
    }

    (2) 방법.
    try { 
            오류가 발생할 수 있는 문장 
    } catch(예외이름1 객체명1) {
            오류 발생 시 실행할 문장
    } catch(예외이름2 객체명2) {
            오류 발생 시 실행할 문장
    }
    
    (3) 방법.
    try {
            오류가 발생할 수 있는 문장             
    } catch (예외이름 객체명) {
            오류 발생 시 실행할 문장
    } finally {
            예외 발생여부에 상관없이 무조건 실행할 문장
    }

    (4) 방법
    try {
            오류가 발생할 수 있는 문장
    } catch)(예외이름1 | 예외이름2 객체명) {
            예외이름 둘 중 하나라도 발생 시 실행할 문장
    }
