컬렉션 프레임워크(Collection Framework)
        많은 데이터를 쉽고 효과적으로 
        관리할 수 있는 표준화된 방법을 
        제공하는 클래스의 집합

컬렉션 프레임워크의 주요 인터페이스
    1. 리스트(List) extends Collection
        구현 클래스
            - Vector
            - ArrayList

    2. 셋(Set) extends Collection
        구현 클래스
            - HashSet

    3. 맵(Map) 
        구현 클래스
            - HashMap
            - TreeMap

List와 Set 인터페이스의 공통요소를 Collection 인터페이스에서 정의하고 
두 인터페이스는 그것을 상속받는다.
