🎁. 제네릭(Generic) : 포괄적인, 총칭의
- 자료형을 정하지 않는 기법(외부에서 지정)
✔. 표기방법 : <대문자>
✔. 예시 :
    <E> - Element
    <K> - Key
    <N> - Number
    <T> - Type
    <V> - Value      
    
1. Generic class 
    - 클래스 내부에서 사용될 자료형을 지정한다. 

2. Generic Method 
    - 클래스에 제네릭을 선언하지 않고 메서드에만 제네릭을 선언한다.
    - 메소드의 매개변수 또는 리턴타입이 제네릭이라면 
    리턴타입 앞에도 제네릭을 반드시 선언해야 한다

3. Generic Interface


싱글톤 패턴 : 단 한개의 객체만은 가지고 있는 클래스를 싱글톤 패턴으로 구현한다.
