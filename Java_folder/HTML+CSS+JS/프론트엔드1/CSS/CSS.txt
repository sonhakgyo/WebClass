CSS(Cascading Style Sheets) 스타일이란?
    
head 태그 안에서 선언문
    선택자   {   속성    :  값;   }
    Selecotr { Property : Value; }

스타일문법 종류
    · 인라인 스타일
    · 내부 스타일시트
        - <STYLE> 태그를 사용하여 HTML 문서에 바로 적용하는 방법
    · 외부 스타일시트
        - 외부에 작성된 .CSS 문서 즉, 스타일시트 파일을 연결하는 방법
    
공간 분할 태그
    블록
        - 박스와 같이 쌓이면서 나타나는 성격/화면 전체를 사용하는 태그
    인라인
        - 글자와 같이 옆으로 나열되며 나타나는 성격/내용의 폭값, 화면 일부를 차지하는 태그
        - 'width, height, margin-top, margin-bottom'은 적용되지 않는다.


공간 분할 태그 스타일

    100%와 auto의 차이
        · width : 100%
            - (비율)%, 부모 기준에 따른 퍼센트 비율로 적용한다.
            - 추가되는 요소를 포함하지 않으며 넘치게 만들 수도 있다.
        · width : auto 
            - 부모 기준으로 꽉 차게 적용한다.
            - 추가되는 요소를 함께 가져가며 폭값을 유지하기 위해 노력한다.

    패딩과 마진
        - 옵션 적용순서는 동일하다. 단 한가지 마진에는 auto 옵션이 있다.
        - ★ margin : 50px auto; 블록 요소의 중심에 배치한다.
