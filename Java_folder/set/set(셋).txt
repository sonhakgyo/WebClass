Set 
    - Set은 집합이다.
    - Set은 인터페이스다.
    - Set을 구현한 대표적인 클래스가 HashSet이다.
    - 순서가 없기 때문에 ArrayList보다 검색속도가 빠르다.
    - 집합은 중복집합이 아니라면 중복되는 원소를 포함할 수 없다.
    - 값(객체, 데이터)을 중복해서 저장할 수 없다.
      저장된 값들은 인덱스가 없기 때문에 저장순서가 고정되어 있지 않다.
    - 함수
        size()     : 저장된 값의 수
        add()      : 값 추가
        iterator() : 검색          ++ 앞으로 map에서 같이 쓰일 예정이다.
        remove()   : 값 삭제

HashSet
    - Set 컬렉션을 구현하는 대표적인 클래스

Set 생성
    - Set<E> 객체명 = new HashSet<E>()
