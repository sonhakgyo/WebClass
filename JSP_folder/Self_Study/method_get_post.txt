메서드 전송방식 method = "GET || POST" 

    GET 전송방식 : URL(주소)에 파라미터를 추가하여 요청하는 방식
        http://주소?이름=값&이름=값&이름=값···
        - 길이 제한이 있다.
        - 어떠한 값이 전달되는지 사용자가 볼 수 있다.
        - <form ··· method="get">

    POST 전송방식 : HTTP request의 body(본문)에 파라미터를 추가하는 방식
        - 길이 제한이 없다.
        - 어떠한 값이 전달되는지 사용자가 볼 수 없다.
        - <form ··· method="post">

각각 언제 쓰는가 
    GET  : 조회(게시판 게시글, 인터넷 뉴스)
    POST : 서버 송신(회원가입, 게시판 등록/수정), 비밀번호 등의 보안이 필요한 파라미터
