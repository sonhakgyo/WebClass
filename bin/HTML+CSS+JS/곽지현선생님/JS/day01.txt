4. 연산자
    4-1. 대입연산자(=) -> 우항의 값을 좌항의 변수에 대입
    4-2. 비교연산자(==) -> 비교값의 결과를 boolean형태로
            리턴해 준다.
    4-3. 일치연산자(===) -> 일치 연산자로 ===는 좌항과 우항이 '정확', '엄격'하게 같을 때 true, 
                            다르면 false가 된다. 
                            javascript, php는 비슷한 것을 같은 것으로 간주하기도 한다. 
                            따라서 '==='를 사용할 것을 권장한다.

    - undefined : 값이 정의(할당)되어 있지 않다.
    - null : 없는(null) 값이 정의되어 있다.

    4-4. 부정(!=, !==) -> !는 부정을 의미한다. 
                '!=='는 정확하게 같지 않다.

    4-5. 크기연산자(>,>=,<,<=) -> boolean 으로 리턴
